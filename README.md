# 🌐 **Core Ecosystem (Communities + AI + Social Layer)**

Prenuers is built as a **multi-layered ecosystem combining social media + startup building + AI guidance**.

---

## 📱 **1. Instagram-like Social Ecosystem (Core Layer)**

This is the **primary engagement engine** of the platform.

### 🔹 Feed System

* Infinite scroll feed
* Content types:

  * 💡 Idea Posts
  * 🚀 Startup Progress
  * 🎤 Pitch Videos
  * 📢 Announcements

### 🔹 Post Features

* Text + Image + Video support
* Tag users / team members
* Add startup association

### 🔹 Engagement Actions

* ❤️ Like
* 💬 Comment (threaded)
* 🔁 Share
* 📌 Save
* 👀 View count

### 🔹 Smart Actions (Startup-specific)

* “Join Team” (for idea posts)
* “Request Validation”
* “Connect with Founder”

### 🔹 Permission Control

* Role-based posting visibility
* Mentor-only comments on validation posts
* Investor-only access to funding posts

👉 This layer ensures **high engagement + discovery + collaboration**

---

## 👥 **2. Founder Community (Student Layer)**

### Features:

* Post and explore startup ideas
* Find co-founders
* Build teams across schools
* Share progress updates
* Participate in discussions

### Unique Value:

* Turns passive users → active builders

---

## 🧠 **3. Mentor Validation Community**

### Features:

* Only verified mentors
* Access to **idea validation dashboard**
* Structured feedback system:

  * Problem clarity
  * Feasibility
  * Market scope
* Status system:

  * Approved
  * Needs Improvement
  * Rejected

### Permissions:

* Can comment on validation posts
* Can approve/review ideas

👉 Ensures **quality control of ideas**

---

## 💰 **4. Investor Community**

### Features:

* Discover startups
* Filter by:

  * Stage
  * Category
* View pitch decks
* Track startup progress

### Actions:

* Connect with founders
* Show funding interest
* Participate in funding rounds

👉 Enables **early-stage funding ecosystem**

---

## 🤖 **5. AI Startup Assistant (Core Differentiator)**

### Capabilities:

* Idea validation (instant feedback)
* Suggest improvements
* Help with:

  * Naming startup
  * Business model
  * Growth strategy
* Explain concepts (learning support)

### UI:

* Floating chatbot
* Full chat screen
* Suggested prompts:

  * “Validate my idea”
  * “How to get users?”
  * “Create a pitch”

### Advanced (Future):

* AI co-founder suggestions
* AI pitch generator
* Personalized learning

👉 Reduces dependency on mentors → enables **scalability**

---

## 💬 **6. Community Chat System**

### Types:

* 1:1 Chat
* Team Chat
* Mentor–Student Chat
* Founder–Investor Chat

### Features:

* Real-time messaging
* Media sharing
